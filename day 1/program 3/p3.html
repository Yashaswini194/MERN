HTML Block-Level and Inline Elements

Understanding Block-Level and Inline Elements

Block-level Element: <div>

Block-level elements take up the full width of their container and typically start on a new line. They are used for larger structural sections of a webpage.

Block-level Element: <section>

This is an example of a <section> element. It is a block-level element used to define sections of content on a page.

Inline Elements:

Inline elements only take up as much width as necessary and do not cause a new line to be created. Examples of inline elements include <span> and <a>.
